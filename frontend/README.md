🐾 Pawsome – Pet Selling Website

Pawsome is an online platform designed to help people discover, post and view pets easily. Users post a picture of the pet and the amount it's valued at.

✨ Features

🐶 Browse Pets – View a wide range of pets such as dogs, cats, birds, and more.

📄 Pet Details Page – View photos, descriptions, health details, and seller information.

🛠️ Tech Stack

The project can be built with the following technologies:

Frontend: React, Tailwind

Backend: Node.js / Express
Database: PostgreSQL, NeonDB
